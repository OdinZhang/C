自增1（自减1）运算符

自增1（自减1）运算符

x += 1;和x++;等价

++和--叫自增1和自减1运算符

前缀和后缀的区别：

y = x++; /*相当于y = x; x += 1;*/
y = x--; /*相当于x += 1; y = x;*/

[danger] 注意

如果有

int i = 5;
printf("%d",i++);

在不同的系统中会输出不同的结果，为6或5，所以最好不要这么写
