C程序基本认识

C程序的基本框架为：

#include "stdio.h"
main()
{
    <语句系列>
}

其中#include "stdio.h"为编译预处理命令，位于程序开始，命令后无分号

main()是C程序的主函数，任何C程序有且仅有一个主函数，都从主函数处开始执行，main是函数名，()表示这个函数没有参数，“{”和“}”括起来的部分叫函数体，分别表示函数体的开始与结束

语句用于描述对象以及对对象的操作，语句写在函数体中，以分号结尾

一行可以写一个或多个语句
