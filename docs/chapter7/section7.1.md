# 枚举类型

#### 1. 枚举类型定义的一般形式

```c
enum 枚举名
{
	枚举常量表列
}；
```

#### 2. 枚举变量的定义

`enum 枚举名 变量名表列;`

#### 3. 注意事项

1. 每一个枚举常量的值取决于在定义时排列的先后顺序，序号从0开始，若想改变枚举常量的值，须从定义时开始指定，如：

   `enum char{a = 1,b,c}svalue`则指定a为1，b为2，svalue是枚举变量

2. 一个枚举变量的值只能是这几个枚举常量之一，可以将一个枚举变量赋给一个枚举变量，但不能将整数赋给他

3. 若想将整数值赋给枚举变量需做强制转换，如：

   `svalue = (enum star)2;`
